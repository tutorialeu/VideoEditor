# 🎬 VideoEditor v1.0.0.0

A lightweight, high-performance video editing suite built with **C# (.NET Framework 4.8)**, **NAudio**, and **FFmpeg**. Designed for precision and ease of use.

![Video Editor UI](https://img.shields.io/badge/Release-v1.0.0.0-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

---

## 🚀 Key Features

- **Multi-Track Timeline**: Layer video and audio with ease.
- **Frame-Accurate Clipping**: Split and cut videos with millisecond precision.
- **High-Precision Audio Sync**: Silky smooth playback synchronization using the NAudio engine.
- **Smart Export**: Advanced concatenation engine with real-time progress reporting.
- **Automatic Muting**: Integrated "Nuclear Mute" for the video previewer, ensuring clean, single-source audio.
- **Project Persistence**: Save and load your editing sessions.

## 🛠️ System Requirements

- **Windows 10/11**
- **.NET Framework 4.8**
- **WMP (Windows Media Player)** components installed.

## 📥 Getting Started

1.  **Download** the latest release from the [Releases](https://github.com/tutorialeu/VideoEditor/releases) page.
2.  **Extract** the contents of the ZIP file to your preferred location.
3.  **Run** `VideoEditor.exe`.
4.  **Import** videos and audios from the "File" menu or drag and drop onto the timeline.
